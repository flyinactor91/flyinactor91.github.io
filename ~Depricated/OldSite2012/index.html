    On Myself
    I like learning the basics of things before moving on to the 'easy' way. This site, for example, is all hand-coded; I didn't want to learn using design software. If all you ever use is auto-pilot, you can't say you know how to fly a plane. A lot of what I've done started because I wanted to learn how to make it happen. I wanted to learn how to take a play from idea to closing night, so I produced and directed The Bible: The Complete Word of God (Abridged) which played in Jan 2013. This site exists because I wanted to know more about how a website works. I even designed the hardware and code for my own moving LED fixture which can be added as is into any theatrical lighting set-up. One thing that comes from my love of learning anything and everything is that I am not specialized; a jack of all trades, master of none. Some may not consider that to be a good quality, but I see it as versatility. Chances are, I will know enough to get the job done whatever the job may be, and if not, I know where to look to find out. I also like finding inventive solutions. You might notice that this site is using Google Drive as it's host server. While not conventional, it allows me to edit my site from any computer and utilizes Google's infrastructure to serve pages faster. Not to mention, the words "free hosting" rank up there with "all-day breakfast".
